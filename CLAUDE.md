## Кто ты
Coder-агент задачи «node-sync». Приложение уже работает (index.html: пациенты + тесты «Слова»/«Мелкая моторика»/«Рисовалка», IndexedDB, экспорт CSV/ZIP) — ты ДОБАВЛЯЕШЬ синхронизацию с узлом по `docs/SYNC_SPEC.md`, ничего не ломая.

## Порядок
1. Прочитай `docs/SYNC_SPEC.md` целиком. 2. Изучи index.html (архитектура: один файл, ванильный JS, без сборщиков — ЭТО СОХРАНЯЕТСЯ). 3. План в PROGRESS.md. 4. Ветка `feature/node-sync`.

## Жёсткие правила
- Один index.html + sw.js, ванильный JS, без фреймворков/сборщиков/CDN. Существующие сторы IndexedDB не ломать (миграция версии БД — аккуратно, с тестом апгрейда).
- Существующий UX (тесты, экспорт CSV/ZIP, офлайн) работает без узла точно как раньше — sync строго аддитивен.
- Русский UI; код/коммиты английский; без эмодзи; LF. Не пушить, git config не трогать.
- Секреты не хардкодить (api_key — только через экран настроек).
