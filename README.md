# Тренажёры — учёт прохождений

Веб-приложение (работает в браузере, без установки и без интернета после первого открытия).
Хранит пациентов и три теста: «Слова», «Мелкая моторика», «Рисовалка».
Все данные лежат локально в браузере (IndexedDB). Экспорт — CSV и ZIP.

## Файлы
- `index.html` — само приложение (можно запускать отдельно)
- `manifest.webmanifest`, `sw.js`, `icon-*.png` — для установки как приложение и офлайн-режима

## Быстрый запуск на ПК (для проверки)
В этой папке выполните:

    python -m http.server 8000

Откройте в браузере: http://localhost:8000
(микрофон для теста «Слова» работает на localhost — это «защищённый контекст»).

## Запуск на планшете/телефоне
Микрофон, стилус и установка как приложение требуют HTTPS **или** localhost.
Самый простой способ — выложить папку на бесплатный HTTPS-хостинг (GitHub Pages),
либо запустить локальный сервер прямо на планшете (Termux). Подробности — в ответе ассистента.

## Стилус (планшет Huawei MatePad Air, DBY2-W09)
Нужен активный стилус Huawei M-Pencil (3-го поколения; 2-е тоже подходит).
Обычная ёмкостная «палка» не даёт отличить перо от ладони — отключения ладони не будет.
